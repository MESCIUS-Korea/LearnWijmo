데이터 공급자 | Data Provider

CollectionView 클래스는 데이터 항목의 배열을 포함하는 sourceCollection을 사용합니다. 이 컬렉션은 일반적으로 서버에서 데이터를 검색하는 http 요청으로 채워집니다.

경우에 따라 서버에서 제공하는 기능을 활용하기 위해 CollectionView 클래스를 확장하는 것이 좋습니다. 예를 들어 Wijmo에는 OData 프로토콜을 활용하여 서버에서 정렬, 필터링 및 페이징을 제공하는 ODataCollectionView가 포함되어 있습니다.

다음은 인기있는 데이터 공급자를 처리하기 위해 CollectionView 클래스를 확장하는 방법을 보여주는 샘플 및 데모에 대한 링크입니다.

  • OData : wijmo.odata.ODataCollectionView 클래스를 사용하여 OData 서비스에 연결하는 방법을 보여줍니다.
  • Virtual OData : wijmo.odata.ODataVirtualCollectionView' 클래스를 사용하여 OData 서비스에 연결하고 주문형 데이터를 표시하는 방법을 보여줍니다.
  • Breeze : BreezeJS 데이터 라이브러리와 함께 작동하는 CollectionView 클래스를 구현하는 방법을 보여줍니다.
  • Firebase : Firebase 데이터베이스에서 작동하는 CollectionView 클래스를 구현하는 방법을 보여줍니다.
