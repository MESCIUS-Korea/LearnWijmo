정렬 | Sorting

.NET에서와 마찬가지로 CollectionView 클래스는 정렬을 지원하는 ICollectionView 인터페이스를 구현합니다.
정렬을 사용하려면 하나 이상의 SortDescription 객체를 배열에 추가하고 CollectionView.sortDescriptions 속성에 전달합니다.
이렇게 하면, SortView.items 속성에서 정렬 된 결과를 가져올 수 있습니다.

SortDescription 객체는 유연하며 오름차순 또는 내림차순으로 값을 기준으로 데이터를 정렬 할 수 있습니다.

아래 예제에서 그리드 머리글의 속성을 클릭하면 해당 속성별로 오름차순으로 정렬됩니다. 속성을 내림차순으로 정렬하려면 다시 클릭하십시오.
