CollectionView 유효성 검사 | Validation

CollectionView에는 유효성 검사 지원을 제공하는 getError 속성이 있습니다. 이를 사용하려면 getError를 유효성을 검사할 데이터 항목과 유효성 속성을 포함하는 두 개의 매개 변수를 취하는 함수로 설정하고 오류 조건을 설명하는 문자열을 반환합니다 (오류가없는 경우 null).

getError 속성은 값 자체 (예 : min, max, required, pattern 등)를 기반으로하는 기본 HTML5 유효성 검사를 능가합니다. 여러 속성과 관련된 조건을 지정할 수 있습니다.

getError 속성을 사용하면 항목 편집에 사용되는 UI 대신 유효성 검사 논리를 컬렉션 자체에 포함 할 수 있습니다. 그런 다음 동일한 양식을 입력 양식이나 FlexGrid와 같은 컨트롤에서 사용할 수 있습니다.
