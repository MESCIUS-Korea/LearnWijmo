CollectionView 시작하기 | Getting Start

Wijmo는 강력하고 익숙한 데이터 계층을 기반으로 견고한 인프라를 갖추고 있습니다. 주요 데이터 바인딩 인터페이스는 ICollectionView입니다. Wijmo에는 ICollectionView를 구현하는 여러 클래스가 포함되어 있습니다. 가장 기본적인 것은 데이터 소스로 일반 JavaScript 배열을 사용하는 CollectionView입니다.

CollectionView 클래스는 다음 인터페이스를 구현합니다.

  • ICollectionView : 현재 레코드 관리, 사용자 지정 정렬, 필터링 및 그룹화를 제공합니다.
  • IEditableCollectionView : 항목 편집, 추가 및 제거를위한 메소드를 제공합니다.
  • IPagedCollectionView : 페이징을 제공합니다.

CollectionView 클래스는 데이터의 변경 내용을 추적 할 수 있습니다. 이 기능은 서버에 변경 사항을 제출해야하는 상황에서 유용합니다.

CollectionView 클래스를 사용하려면 먼저 선언하고 일반 배열을 데이터 소스로 전달하십시오. 그런 다음 items 속성을 사용하여 뷰에 액세스합니다.

응용 프로그램에서 CollectionView 클래스를 시작하기위한 단계 :

  1. Wijmo에 대한 참조를 추가하십시오.
  2. FlexGrid의 호스트 역할을하는 마크 업을 추가하십시오.
  3. Javascript를 통해 CollectionView 인스턴스와 FlexGrid 인스턴스를 초기화하십시오.
  4. (선택 사항) CSS를 추가하여 그리드 모양을 사용자 정의하십시오.
