페이징| Paging

CollectionView 클래스는 .NET Framework와 거의 동일한 IPagedCollectionView 인터페이스를 통한 페이징을 지원합니다.
페이징을 사용하려면 IPagedCollectionView.pageSize 속성을 각 페이지에서 원하는 항목 수로 설정하고 페이지를 탐색하기위한 UI를 제공하십시오.

이 예제에서는 JavaScript를 사용하여 페이지 당 10 개의 항목을 표시합니다. 텍스트 상자에서 사용자 정의 할 수 있습니다. 탐색 버튼을 추가하고 버튼 클릭으로 IPagedCollectionView메소드를 호출합니다.

pageIndex 및 pageCount 속성을 사용하여 현재 페이지 및 총 페이지 수를 표시합니다. 첫 번째 텍스트 상자에서 페이지 크기를 사용자 정의 할 수 있습니다. 비우거나 0으로 설정하면 CollectionView는 페이징을하지 않습니다. 그러면 탐색 버튼이 보이지 않게됩니다.
