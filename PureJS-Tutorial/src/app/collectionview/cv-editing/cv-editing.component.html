편집 | Editing

IEditableCollectionView 인터페이스를 구현할 때 CollectionView 클래스는 편집을 지원합니다.

이 샘플은 컬렉션에서 지정된 항목을 업데이트, 추가 및 제거하는 방법을 보여줍니다.

이 샘플에서는 그리드의 행을 선택하고 Edit Detail ... 버튼을 눌러 편집을 시작할 수 있습니다. 팝업 대화 상자에서 편집을 마친 후 확인 버튼을 눌러 업데이트를 커밋합니다.
컬렉션에 새 레코드를 추가하려면 Add... 단추를 누르고 팝업 대화 상자에서 항목 내용을 사용자 지정하십시오. 그런 다음 확인 버튼을 눌러 추가를 커밋합니다.
레코드를 업데이트 / 추가하지 않으려면 대화 상자에서 취소 버튼을 누르십시오.
행을 선택하고 Delete 버튼을 누르면 컬렉션에서 레코드를 제거 할 수 있습니다.

업데이트, 추가 및 제거 후, 그리드는 추적 된 항목 배열에 따라 새로 고쳐집니다.

Result와 함게 HTML, JS 코드도 확인이 가능합니다.
