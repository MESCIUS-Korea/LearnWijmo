차트 툴팁 | Tooltips

FlexChart에는 툴팁(Tooltip)이 내장되어 있습니다. 기본적으로 컨트롤은 데이터 포인트에서 마우스를 터치하거나 마우스를 가리킬 때 툴팁을 표시합니다.

툴팁 컨텐츠는 다음 매개 변수를 포함 할 수있는 템플릿을 사용하여 생성됩니다:

  • seriesName : 차트 요소가 포함 된 계열의 이름입니다.
  • pointIndex : 계열 내의 그래프 엘리먼트의 인덱스입니다.
  • x : 차트 요소의 x 값입니다.
  • y : 차트 요소의 y 값입니다.

기본적으로 툴팁 템플릿은 <b>{seriesName}</b><br/> {x} {y}로 설정되어 있으며 아래의 차트에서 어떻게 작동하는지 확인할 수 있습니다. 이 예에서는 툴팁 템플릿을 <b>{seriesName}</b><img src = 'resources/{x}.png'/><br/> {y} 로 설정하여 나라의 국기가 나오게 합니다.
