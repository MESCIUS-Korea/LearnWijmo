차트 컨트롤 생성 | Getting Start

JavaScript 애플리케이션에서 FlexChart를 시작하기위한 단계 :

  1. Wijmo에 대한 참조를 추가하십시오.
  2. FlexChart의 호스트 역할을하는 마크 업을 추가합니다.
  3. JavaScript와 itemSource 속성을 통해 FlexChart를 초기화하십시오.
  4. 하나 이상의 데이터 시리즈를 만들고 각각을 FlexChart의 시리즈 컬렉션에 추가하십시오.
  5. (선택 사항) CSS를 추가하여 차트 모양을 사용자 정의합니다.

Result와 함게 HTML, JS 코드도 확인이 가능합니다.
