바인딩 | Binding

대부분의 시나리오에서 FlexGrid는 서버에서 로드되거나 코드에서 생성된 데이터를 표시합니다.이 데이터는 객체의 배열로 표현되며 그리드는 itemsSource 속성을 통해이 배열에 바인딩됩니다.

기본적으로 itemsSource 속성을 설정하면 그리드가 데이터 개체를 검색하고 데이터 항목의 각 속성에 대한 열을 자동으로 생성합니다.

Column 자동 생성은 간단한 데이터 소스 또는 응용 프로그램 테스트/개발 중에는 잘 작동합니다.
그러나 대부분의 경우 자동 열 생성을 해제하고 코드에서 열을 지정해야합니다.이 방법으로 열이 표시되는 순서, 너비, 형식, 헤더 등을 완벽하게 제어 할 수 있습니다.

각 열에는 셀에 표시되어야하는 데이터 항목의 속성을 결정하는 바인딩 속성이 있습니다.
