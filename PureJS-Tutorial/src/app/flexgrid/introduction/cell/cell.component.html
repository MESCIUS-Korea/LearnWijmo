셀 | Cells

FlexGridM 모듈은 Row 클래스와 Column 클래스를 정의하지만 Cell 클래스는 정의하지 않습니다.이는 셀이 Row와 Column의 교차 부분을 나타 내기 위해 만들어진 DOM 요소 일 뿐이기 때문입니다.

렌더링 할 때 FlexGrid는 viewRange의 행과 열을 스캔하고 각 조합에 대해 셀 요소를 생성합니다.셀 요소는 행에서 제공하는 dataItem과 열에 의해 제공되는 바인딩을 기반으로 다른 속성과 함께 만들어집니다.

formatItem 이벤트

경우에 따라, 여러분은 데이터에 기반하여 셀 요소를 커스터마이즈 하거나, 사용자 정의 셀 요소를 생성하거나, 또는 더 이상 바인딩(계산된 컬럼, 언바운드 그리드)을 사용하고 싶지 않을때가 있을 수 있습니다.

이 경우 formatItem 이벤트를 사용하여 셀 요소를 만들거나 수정하십시오.
예를 들어, 아래 표는 1,000 단위 미만의 판매량을 빨간색으로 표시하고 5,000 단위 이상의 매출을 녹색으로 표시합니다.

Getting / setting 셀 데이터

대부분의 경우 바인딩 원본 개체에서 직접 데이터를 수정하고 그리드에 결과를 표시합니다. 그러나 경우에 따라서는 그리드를 통해 데이터를 가져 오거나 설정할 수 있습니다.

GridPanel 클래스의 getCellData 및 setCellData 메소드를 사용하여이 작업을 수행 할 수 있습니다.이 메소드는 원시 또는 형식화 된 데이터를 가져오고, 데이터를 설정할 때 데이터 유형을 강제 변환하고, 바인드되지 않은 그리드에서 데이터를 가져 오거나 설정하는 옵션을 제공합니다.
