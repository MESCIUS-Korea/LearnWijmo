Clipboard Support

FlexGrid에는 빌트인 클립 보드 지원 기능이 있습니다.

기본적으로 ctrl + c 또는 ctrl + Insert 키를 누르면 현재 선택 항목이 클립 보드로 복사됩니다.
ctrl + v 또는 shift-Insert를 누르면 클립 보드 내용이 눈금에 붙여 넣어집니다.

클립보드 비활성

autoClipboard 속성을 false로 설정하여 자동 클립 보드 기능을 비활성화 할 수 있습니다. 
flexGrid.autoClipboard = false;

사용자 정의 클립보드

copying, copied, pasting, pasted, pastingCell 및 pastedCell 이벤트를 사용하여 클립보드 동작을 사용자 정의 할 수 있습니다.
