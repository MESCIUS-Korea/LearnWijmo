선택모드 | Selection

기본적으로 FlexGrid를 사용하면 Excel과 마찬가지로 마우스 또는 키보드로 셀 범위를 선택할 수 있습니다.

selectionMode 속성을 사용하면 사용자가 행, 행 범위, 인접하지 않은 행, 단일 셀 또는 아무 것도 선택하지 못하도록 변경할 수 있습니다.

Members

Name Value Description
None 0 사용자는 마우스 나 키보드를 사용하여 셀을 선택할 수 없습니다.
Cell 1 사용자는 한 번에 하나의 셀만 선택할 수 있습니다.
CellRange 2 사용자는 인접한 셀 블록을 선택할 수 있습니다.
Row 3 사용자는 한 번에 한 행을 선택할 수 있습니다.
RowRange 4 사용자는 인접한 행을 선택할 수 있습니다.
ListBox 5 사용자는 인접하지 않은 행을 선택할 수 있습니다.

현재 selectionMode에 관계없이, 그리드는 선택이 변경되면 selectionChanging 및 selectionChanged 이벤트를 발생시킵니다. 그리고 selection 속성을 사용하여 현재 선택을 가져 오거나 설정을 CellRange 값으로 설정할 수 있습니다. 

theGrid.selection = new wijmo.grid.CellRange(0, 0, 1, 1);

ListBox 모드에서 사용자는 Ctrl + 클릭을 사용하여 개별 행을 선택할 수 있으며 행의 isSelected 속성을 사용하여 행이 선택되었는지 여부를 확인할 수 있습니다. 

theGrid.rows[index].isSelected = true;

FlexGrid에는 빌트인 클립 보드 지원 기능이 있습니다. ctrl + c 키를 눌러 현재 선택 항목을 클립 보드에 복사하십시오.
