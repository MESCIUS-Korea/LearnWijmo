필터 최적화 | Filter Optimizations

wijmo.grid.filter 모듈은 Excel 스타일의 필터링 UI를 각 열에 추가하는 FlexGridFilter 클래스를 제공합니다. FlexGridFilter 는 value 및 condition 의 두 가지 유형의 필터를 제공합니다.

value 필터 편집기에는 사용자가 선택할 수있는 고유 한 값 목록이 포함되어 있습니다. 그리드에 많은 양의 데이터가 포함되어 있으면 이 고유한 값 목록을 생성하는 데 시간이 걸릴 수 있습니다. 또한 목록에 너무 많은 값이 포함되어 있으면 매우 유용하지 않습니다 (이러한 경우 조건 필터가 더 적합 할 수 있음)

이 샘플은 세 가지 방법으로 value 필터를 최적화하는 방법을 보여줍니다.

  1. uniqueValues :'Rating'열 필터는 열에서 가능한 고유 값 목록을 지정하므로 필터는 목록을 작성하기 위해 데이터를 검사할 필요가 없습니다.
  2. maxValues :'Sales'열 필터는 목록에 최대 20 개의 값만 표시되도록 지정합니다. 편집기에서 필터 필드를 사용하여 관심있는 값을 선택할 수 있습니다.
  3. filterType :'Expenses'열 필터에는 조건(Condition) 필터 타입을 지정합니다. 이 열에는 값 필터가 표시되지 않습니다.
