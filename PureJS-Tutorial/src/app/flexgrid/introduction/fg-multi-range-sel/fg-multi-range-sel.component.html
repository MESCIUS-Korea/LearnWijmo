FlexGrid에는 사용자가 인접하지 않은 행 범위를 선택할 수있는 ListBox 선택 모드가 있습니다. 그러나 사용자가 여러 개의 인접하지 않은 셀 범위를 선택할 수있는 기본 제공 선택 모드는 없습니다.
