헤더 병합 | Header Merge

FlexGrid에서는 기본적으로 컨텐츠를 기반으로 셀(Cell) 병합과 같은 방식으로 헤더(Header) 병합를 지원해줍니다.

헤더 병합을 사용하려면 allowMerging 속성 값을 All 또는 AllHeaders 설정하여 병합 할 그리드의 일부 또는 부분을 지정하고 특정 행과 열의 allowMerging 속성을 true로 설정하십시오. 그렇게하면 그리드가 동일한 내용의 셀을 병합하여 데이터를 시각적으로 그룹화합니다. 

//Grid 전체에 대해서 병합(Merge) 허용
flexGrid.allowMerging = wijmo.grid.AllowMerging.All;

//Grid의 전체 헤더 대해서 만, 병합(Merge) 허용
flexGrid.allowMerging = wijmo.grid.AllowMerging.AllHeaders;

사용자 정의 헤더(Header)병합

기본적으로 FlexGrid는 내용을 기반으로 셀에 병합을 적용합니다. 하지만 경우에 따라, 병합에 다양한 형식을 취해야 할 경우가 있습니다. 이때, 사용자 정의 병합을 사용하여 원하는 형식으로 병합을 할 수 있습니다.

wijmo.grid.MergeManager 클래스를 확장하는 클래스를 정의하고 해당 유형의 객체를 그리드의 mergeManager 속성에 할당하면됩니다.
사용자가 병합할 범위를 설정하는 방법들은 아래의 샘플 코드를 참고하시길 바랍니다. 

// 사용자 정의 병합 메니저
var mm = new wijmo.grid.MergeManager(grid);
var rng = new wijmo.grid.CellRange(0, 1, 1, 2);
var rng1 = new wijmo.grid.CellRange(0, 0, 2, 0);
grid.mergeManager = mm;
