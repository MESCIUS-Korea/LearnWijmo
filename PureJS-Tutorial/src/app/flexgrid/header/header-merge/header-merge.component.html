헤더 병합 | Header Merge

FlexGrid에서는 기본적으로 컨텐츠를 기반으로 Cell Merge와 같이 Header를 지원해줍니다.

헤더 병합을 사용하려면 allowMerging 속성 값을 All 또는 AllHeaders 설정하여 병합 할 그리드의 일부 또는 부분을 지정하고 특정 행과 열의 allowMerging 속성을 true로 설정하십시오. 그렇게하면 그리드가 동일한 내용의 셀을 병합하여 데이터를 시각적으로 그룹화합니다. 만약, 해당 행(Row) 이동 기능을 막고 싶다면, 아래와 같은 코드르 통해서 Disable 할 수 있습니다.

flexGrid.allowDragging = "None";

Members

Name Value Description
None 0

사용자는 행이나 열을 이동할 수 없습니다.
Columns 1

사용자는 열을 이동할 수 있습니다.
Rows 2

사용자는 행을 이동할 수 있습니다.
Both Rows | Columns

사용자는 열과 행, 모두를 이동할 수 있습니다.
