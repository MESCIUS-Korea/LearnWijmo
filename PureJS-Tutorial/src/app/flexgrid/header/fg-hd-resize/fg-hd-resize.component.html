헤더 사이즈 고정 | Header Resize

사용자로 하여금, 미리 디자인 해놓은 헤더의 크기를 변경하지 않도록 설정해야할 경우가 있을 것입니다. 이럴 경우, allowResizing 속성 값을 설정해 줌으로써 헤더의 변경을 금지할 수 있습니다.

필터 패널을 사용하기 위해서는 Header 태그에 반드시 Input, Filter, Grouppanel 클래스를 선언해주어야 합니다. 

// 행렬 헤더의 크기 변경을 금지합니다.
flexGrid.allowResizing = wijmo.grid.AllowResizing.None;
