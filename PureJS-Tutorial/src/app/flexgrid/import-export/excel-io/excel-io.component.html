FlexGrid 컨트롤을 XLSX 형식으로 내보내려면 응용 프로그램에 세 개의 추가 모듈을 포함해야합니다.

  1. wijmo.xlsx.js: XLSX 파일을 저장하고 로드하는 일반적인 방법을 제공합니다.
  2. wijmo.grid.xlsx.js: wijmo.xlsx.js 를 사용하여 FlexGrid 컨트롤을 XLSX로 저장하거나 XLSX 파일을 FlexGrid 컨트롤로 로드하는 FlexGridXlsxConverter 클래스가 있습니다.
  3. jszip.js: ZIP 파일 생성, 읽기 및 편집을위한 Javascript 라이브러리.

FlexGrid를 XLSX로 내보내려면 FlexGridXlsxConverter.save 메서드를 호출하여 'book'객체를 가져옵니다. 예를 들어 시트를 추가하거나 이름을 변경하여 책 오브젝트를 저장하기 전에 수정할 수 있습니다. 'book'이 준비되면 저장 메소드를 호출하여 XLSX 파일을 만듭니다.
