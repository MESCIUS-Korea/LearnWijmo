수정가능 트리 그리드 | Editable Tree Grid

FlexGrid의 childItemsPath 를 사용하여 데이터를 트리로 표시하면 그결과 그리드는 기본적으로 읽기 전용이됩니다. 이것은 트리의 모든 행이 GroupRow 이고 그룹 행이 기본적으로 읽기 전용이기 때문에 발생합니다.

트리를 편집 가능하게하려면 loadedRows 이벤트를 처리하여 모든 행의 isReadOnly 속성을 false로 설정하십시오.

열의 isReadOnly 속성 또는 beginningEdit 이벤트를 사용하여 편집 할 수있는 열을 계속 제어 할 수 있습니다.
