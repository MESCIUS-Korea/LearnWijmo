OData | ODataCollectionView

Wijmo에는 OData 소스를 지원하기 위해 CollectionView를 확장하는 ODataCollectionView 클래스가 포함되어 있습니다.

OData 는 OASIS standard RESTful API를 빌드하고 소비하는 모범 사례를 정의합니다.OData는 RESTful API를 구축하면서 비즈니스 로직에 집중할 수 있도록 도와줍니다. 요청 및 응답 헤더, 상태 코드, HTTP 메소드, URL 규칙, 미디어 유형, 페이로드 형식, 쿼리 옵션 등을 정의하는 다양한 방법을 염려할 필요가 없습니다.

ODataCollectionView 클래스를 사용하려면 검색할 필드와 같은 선택적 매개변수에 액세스하려는 테이블의 이름과 데이터 서비스의 URL을 전달하는 새 인스턴스를 만들고 해당 필드에서 필터링, 정렬 및 페이징을 서버 또는 클라이언트에서 수행할지 여부를 지정합니다.

데이터 소스에서 글 쓰기를 지원하는 경우 키 필드도 포함하십시오.

이 예제는 공용 (읽기 전용) OData 원본에서로드 된 Northwind 고객 목록을 보여줍니다.
