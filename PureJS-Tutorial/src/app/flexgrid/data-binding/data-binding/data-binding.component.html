데이터 바인딩 | Data Binding

데이터 바인딩은 FlexGrid를 데이터 소스에 연결하여 item 컬렉션을 보고 편집하는 데 사용할 수 있도록하는 프로세스입니다.

언바운드 (Unbound) Grids

대부분의 그리드는 바운드 모드에서 사용되지만 FlexGrid는 바인딩되지 않은 컨트롤로 사용할 수도 있습니다. 그리드에 행과 열을 추가하고 setCellData 및 getCellData 메서드를 사용하여 각 셀의 데이터를 가져오거나 설정할 수 있습니다.

열 자동생성 (Auto-Generating Columns)

코드로 열(Column) 정의

일부 시나리오에서는 자동으로 열을 생성하지만 일반적으로 표시할 열과 형식을 지정하는 방법을 그리드에 명시적으로 알려야합니다. autoGenerateColumns 속성을 false로 설정하고 그리드의 columns 컬렉션을 채우면됩니다.
