OData와 가상 데이터 | OData and Virtual Data

ODataCollectionView 클래스는 컨트롤을 OData 소스에 연결하는 간단한 방법을 제공합니다. ODataCollectionView 를 만들면 소스에서 데이터로드가 시작됩니다.

on-demand 데이터로드를 제공하기 위해 ODataVirtualCollectionView 가 ODataCollectionView 를 확장합니다. 서버의 데이터를 자동으로로드하지 않습니다. 대신 요청시 setWindow 메서드를 사용하여 데이터 조각 (윈도우)을로드합니다.

아래의 그리드는 두 클래스가 어떻게 작동하는지 보여줍니다. 왼쪽 그리드에 점진적으로로드되는 데이터가 표시되는 것에 주목하십시오. 오른쪽의 표에는 전체 레코드 수가 즉시 표시되지만 아래로 스크롤전엔 데이터가로드되지 않습니다.
