행 이동 / Row Move

FlexGrid에는 기본적으로 사용자 원하는 행(Row)을 Drag&Drop으로 원하는 위치로 이동할 수 있습니다.

만약, 해당 행(Row) 이동 기능을 막고 싶다면, 아래와 같은 코드르 통해서 Disable 할 수 있습니다.

flexGrid.allowDragging = "None";

Members

Name Value Description
None 0

사용자는 행이나 열을 이동할 수 없습니다.
Columns 1

사용자는 열을 이동할 수 있습니다.
Rows 2

사용자는 행을 이동할 수 있습니다.
Both Rows | Columns

사용자는 열과 행, 모두를 이동할 수 있습니다.

Result와 함께 HTML, JS 코드도 확인이 가능합니다.
