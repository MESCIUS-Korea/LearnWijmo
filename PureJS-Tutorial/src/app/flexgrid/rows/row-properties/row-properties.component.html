행 속성 | Row Properties

wijmo.grid.Row 클래스에는 각 행의 모양과 동작을 구성하는 데 사용할 수있는 거의 20 개의 속성이 있습니다.

그리드가 새로 고침(refresh)되면 스크롤 가능한 행이 다시 생성되므로 새 데이터에 바인딩 될 수 있습니다. 이 때문에 행을 만든 후에 시작되는 loadedRows 이벤트에 대한 응답으로 일반적으로 행 속성을 적용해야합니다.

대부분의 행은 Row 클래스의 인스턴스이지만 그룹화된 그리드에는 일반 Row 클래스를 확장하는 GroupRow 객체가 포함될 수 있습니다.

Row 클래스의 가장 중요한 속성은 다음과 같습니다.

  • dataItem: 행에 바인딩 된 데이터 항목에 대한 참조를 포함합니다. 이 속성은 바운드 행을 만들 때 그리드에 의해 설정되며 formatItem 이벤트 처리기에서 자주 사용됩니다. GroupRow 개체의 dataItem 속성에는 일반 데이터 항목이 아닌 행의 Group에 대한 참조가 포함되어 있습니다.
  • isReadOnly: 행을 편집 할 수 있는지 여부를 가져 오거나 설정합니다. 기본적으로 Row 인스턴스는 수정 가능하며 GroupRow 인스턴스는 수정할 수 없습니다.
  • isSelected: 이 속성은 그리드의 selectionMode 속성이 ListBox 로 설정된 경우 중요합니다. 이 경우 사용자는 인접하지 않은 행 집합을 선택할 수 있습니다. 또한 isSelected 속성을 사용하면 개별 행의 선택된 상태를 가져 오거나 설정할 수 있습니다.
  • visible: 행이 표시되는지 여부를 가져 오거나 설정합니다. 이 속성을 사용하여 조건 적으로 행을 숨길 수 있습니다.
  • isVisible: 행이 현재 표시되어 있는지 여부를 나타내는 값을 가져옵니다. 표시된 속성이 true로 설정되어 있어도 행이 축소 된 그룹의 일부이므로 숨겨져있을 수 있습니다.
