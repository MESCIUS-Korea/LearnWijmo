상세보기 행 | Row Details

때로는 행 안에 들어갈 수있는 정보보다 많은 정보를 포함하는 경우가 있을 수 있습니다. FlexGrid에서는 이러한 상황에서 쉽게 그리드에 행 상세정보를 표현할 수 있도로고 상세보기 행 기능을 제공하여 드리고 있습니다.

wijmo.grid.detail 모듈에 포함된 FlexGridDetailProvider 클래스를 사용하면 됩니다.

FlexGridDetailProvider는 행 헤더에 축소/확장 버튼을 추가하고 해당 항목에 대한 추가 세부 정보를 제공하는 데 사용할 수 있는 createDetailCell 메서드를 사용하여 FlexGrid를 확장합니다. 세부 정보는 세부 정보가 확장되면 그리드에 추가 된 '세부 행'에 표시되고 축소되면 숨겨집니다.

FlexGrid의 상세보기 행을 적용하기 위해서는 wijmo.grid.detail.min.js 파일을 header에 선언해야 합니다.

<script src="scripts/vendor/controls/wijmo.grid.detail.min.js"></script>
