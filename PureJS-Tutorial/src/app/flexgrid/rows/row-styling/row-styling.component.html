행 스타일 | Row Styling

기본적으로 그리드는 모든 다른 행에 'wj-alt'클래스를 추가하고 'wijmo.css'파일은이 클래스를 사용하여 행을 번갈아 스타일을 지정합니다. 사용자 정의 CSS를 사용하여이를 비활성화 할 수 있지만 행 스타일을 전혀 적용하지 않으려면 격자의 showAlternatingRows 속성을 false로 설정하는 것이 더 쉽습니다.

FlexGrid는 데이터 정렬, 필터링, 그룹화 또는 편집시 발생하는 데이터 소스가 refresh 될 때마다 행을 다시 생성합니다. 이 때문에, 대부분의 경우에서 행 속성이 값을 유지할 것으로 예상해서는 안됩니다.

행에 사용자 정의 스타일을 적용하려면 그리드에서 행을 다시 생성 할 때마다 발생하는 loadedRows 이벤트가 발생할 때 이 작업을 수행해야합니다.

스타일에 영향을 미치는 행 속성은 다음과 같습니다.

  • cssClass :이 행의 셀에 추가 할 클래스 이름을 지정합니다. 클래스 이름은 CSS 규칙에서 이 행의 셀 스타일을 수정하는 데 사용할 수 있습니다.
  • height : 행의 높이 (픽셀 단위)입니다.
