행 추가/삭제 | Adding and Removing Rows

FlexGrid에서는 자체적으로 그리드에 행을 추가하거나 제거하는 기능을 제공하여 드리고 있습니다.
그렇기 떄문에, 개발자는 별도의 코딩 없이도 기본 삭제/추가 기능을 사용할 수 있습니다.

런타임에 사용자가 행을 추가하거나 제거할 수 있게 하려면 다음의 속성 정보를 설정하십시오:

  • allowAddNew:
    이 속성을 true로 설정하면 그리드 아래쪽에 새로운 행 추가 '템플릿'이 표시됩니다. 사용자는 새 행 템플리트의 셀을 채워 itemsSource에 새로운 행을 추가 할 수 있습니다.
  • allowRemove:
    이 속성을 true로 설정하면 그리드가 키보드의 'Delete'키를 처리하고 선택된 행을 제거합니다. 사용자는 행 머리글을 클릭하여 행을 선택한 다음, 'Delete'키를 눌러 itemsSource 배열에서 바인딩된 항목을 제거 할 수 있습니다. 
    •   var flexGrid = new wijmo.grid.FlexGrid('#flexGrid', {
    itemsSource: data,
    allowAddNew: true,
    allowDelete: true,
  });
  • newRowAtTop:
    이 속성을 true로 설정하면 그리드의 아래쪽이 아닌 그리드의 위쪽에 새 행 '템플릿'이 표시됩니다. 
    flexGrid.newRowAtTop = true;
