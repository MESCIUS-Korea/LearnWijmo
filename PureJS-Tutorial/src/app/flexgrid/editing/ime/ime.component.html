IME | Input Method Editor

IME는 Input Method Editor 의 약자입니다.

IME를 사용하면 키보드 입력이나 마우스 이동과 같은 모든 데이터를 입력으로받을 수 있습니다.이 방법으로 사용자는 입력 장치에없는 문자 및 기호를 입력 할 수 있습니다. IME는 일본어, 중국어, 한국어 및 대만어와 같은 언어에 대해 의무 사항입니다.

FlexGrid는 imeEnabled 속성을 통해 IME를 지원합니다.예를 들어 아래의 그리드에는 imeEnabled 가 true로 설정되어 있습니다.

키보드 언어를 일본어 또는 중국어로 설정하고 IME를 활성화하면, 그리드는 해당 설정을 따르며 셀에 직접 입력하여 선택한 IME 모드를 사용하여 셀을 편집 할 수있게합니다.
