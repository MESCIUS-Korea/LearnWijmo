정렬 | Sorting

그리드는 CollectionView 데이터 소스를 통해 정렬을 지원합니다.

기본적으로 모든 열 머리글을 클릭하면 클릭 한 열을 기반으로 데이터가 정렬됩니다. 같은 열 머리글을 다시 클릭하면 정렬 순서가 바뀝니다.
Ctrl 키를 누른 상태에서 열 머리글을 클릭하면 정렬이 제거됩니다.

그리드의 showSort 및 allowSorting 속성을 사용하여 정렬 비헤이비어를 사용자 정의 할 수 있습니다.
열의 allowSorting 속성을 false로 설정하여 특정 열의 정렬을 비활성화 할 수 있습니다 

flexGrid.allowSorting = false;
