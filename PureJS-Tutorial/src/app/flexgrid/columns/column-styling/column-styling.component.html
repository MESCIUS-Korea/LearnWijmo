열 스타일 | Column Styling

FlexGrid 열은 스타일에 영향을주는 몇 가지 속성을가집니다:

  • cssClass : 이 열의 셀에 추가 할 클래스 이름을 지정합니다. 클래스 이름은 CSS 규칙에서이 열의 셀 스타일을 수정하는 데 사용할 수 있습니다.
  • align : 셀에있는 텍스트의 수평 정렬을 지정합니다. 이 속성은 기본적으로 null로 설정되어 있으므로 눈금에서 열의 dataType을 기준으로 맞춤을 선택합니다 (숫자는 오른쪽 맞춤, 부울 값은 가운데 맞춤 및 다른 유형은 왼쪽 맞춤). 기본 정렬을 재정의하려면이 속성을 'left' 'right'또는 'center'로 설정하십시오.
  • wordWrap : 셀의 텍스트를 셀 내에서 줄 바꿈 할 수 있는지 여부를 지정합니다. autoSizeRows 메서드를 호출하지 않으면 격자가 셀 내용에 맞게 행 높이를 조정하지 않습니다.

cssClass는 세 가지 속성 중에서 가장 강력하고 유연하지만 align 및 wordWrap은 간단하고 편리합니다.
또한, align과 wordWrap은 헤더를 포함한 모든 셀에 적용되는 반면 cssClass는 스크롤 가능한 셀에만 적용됩니다.
