데이터 맵 | DataMap

많은 경우, 셀에 실제로 그리드에 저장된 값과 다른 값이 표시되도록 열에 값을 매핑해야 할 수 있습니다.

예를 들어, 0부터 3까지 'rating(등급)' 컬럼이 있다면, 아마 여러분은 이를 'Height(상)', 'Middle(중)', 'Low(하)'로 표현하고 싶을 떄가 있을 것입니다.
또는, ID가 들어있는 'customer(고객)' 컬럼이 있다며, 이를 ID 대신 '이름'으로 보여줘야 할 떄가 있을 것 입니다.

formatItem 이벤트를 사용하여 두 작업을 모두 수행 할 수 있지만 FlexGrid가 더 나은 대안인 DataMaps를 제공합니다.
열의 dataMap 속성을 DataMap의 인스턴스로 설정하면 그리드에서 다음을 수행합니다.

  • 각 데이터 항목의 표시 값 조회하고,
  • 셀을 편집 할 때 유효한 항목이있는 드롭다운 목록을 제공합니다.

예를 들어 아래 그리드에는 고객 이름과 ID를 연결하는 DataMap이있는 'Customer(고객)'열이 있습니다. 데이터 소스에는 고객 ID가 들어 있지만 그리드에는 ID 대신 이름이 표시되며 각 항목에 대한 고객을 선택하기위한 드롭-다운 목록이 제공됩니다.

그리드는 또한 'Country(국가)'열에 dataMap을 지정합니다. 이 경우 Map은 국가 이름이있는 문자열 배열입니다. 실제 매핑은 없지만 여전히 드롭-다운 목록을 가져옵니다. 사용자는 목록에없는 국가를 입력 할 수 없습니다.
