그룹핑 | Grouping

여러분은 FlexGrid에서 여러가지 방법으로 데이터를 그룹화 할 수 있습니다.

기본 CollectionView 그룹핑 - 속성 값 이용


그리드의 collectionView.groupDescriptions 속성을 설정하여, 그리드 초기화 시에 하나 이상의 속성으로 데이터를 그룹화 할 수 있습니다.
이것은 쉽게 할 수 있지만 특정 종류의 그룹화 UI를 원한다면 직접 구현해야합니다. 

  // 그리드 초기화 및 그룹핑 초기화
   var flexGrid = new wijmo.grid.FlexGrid('#flexGrid', {
   itemsSource: new wijmo.collections.CollectionView(data, {
   groupDescriptions: [
                         'country',
                         'product'
                       ]
     })
   });

기본 CollectionView 그룹핑 - 함수 이용


그리드의 collectionView.groupDescriptions.push() 함수를 통해, 그리드 초기화 이후에, 사용자가 원하는 방식으로 그룹을 재설정 할 수 있습니다. 

  //County 그룹 디스크립션 설정
  cv.groupDescriptions.push(new wijmo.collections.PropertyGroupDescription('country'));

  //Product 그룹 디스크립션 설정
  cv.groupDescriptions.push(new wijmo.collections.PropertyGroupDescription('product'));

기본 CollectionView 그룹핑 - 그룹패널 사용


Wijmo의 wijmo.grid.grouppanel은 GroupPanel(그룹패널) 컨트롤을 제공해줍니다.
이를 통해, 사용자는 그룹패널에 헤더를 드래그해 놓는 것 만으로도 손쉽게 그룹핑을 할 수 있습니다.

FlexGrid의 그룹패널(GroupPanel)을 적용하기 위해서는 wijmo.grid.grouppanel.min.js 파일을 header에 선언해야 합니다. 

<script src="scripts/vendor/controls/wijmo.grid.grouppanel.min.js"></script>

GroupPanel 컨트롤을 사용하려면 열 헤더를 패널로 드래그하여 그룹을 만듭니다. 그룹을 새 위치로 드래그하거나 그룹을 제거 할 수도 있습니다.

사용자 정의 그룹헤더


FlexGrid의 groupHeaderFormat의 속성을 변경하여, 사용자가 보여주고 싶은 형식으로 그룹헤더를 정의할 수 있습니다. 

flexGrid.groupHeaderFormat = '{name}: {value}';
