사용자 정의 집계 | Custom Aggregates

FlexGrid 열은 전체 그리드 또는 각 그룹에 대한 데이터 요약을 표시 할 수있는 집계 속성을 가지고 있습니다.

그러나 어떤 경우에는 집계 속성이 충분히 유연하지 않으므로 사용자 정의 코드를 사용하여 집계를 계산해야 할 수도 있습니다.

아래 표에는 '매출'과 '비용'의 차이를 보여주는 '수익'열이 있습니다.

'Profit'열은 formatItem 이벤트에서 계산됩니다. 일반 데이터 항목의 이익은 실제 데이터 항목을 기반으로합니다. 그룹의 이익은 group의 getAggregate 메소드를 사용하여 계산됩니다.
