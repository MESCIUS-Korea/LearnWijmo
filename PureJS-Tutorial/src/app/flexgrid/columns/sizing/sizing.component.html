크기변경 | Sizing

그리드를 생성할때 사용자가 원하는 열 너비를 설정할 수 있습니다. 이후에는 사용자가 마우스를 사용하여 자유자제로 열의 너비를 조정할 수 있습니다.
Excel에서과 마찬가지로 열 헤더의 가장자리에서 드래그 하거나, 가장자리를 두 번 클릭하여 내용에 맞게 자동 크기를 조정할 수 있습니다.

사용자가 열의 크기를 조정하지 못하도록하려면, 그리드의 allowResizing 속성을 'None'으로 설정하거나, 열의 allowResizing 속성을 'false'로 설정하여 특정 열의 크기를 조정하지 못하게 할 수 있습니다. 

var flexgrid = new wijmo.grid.FlexGrid('#flexgrid', {
  deferResizing: true, // show just a marker while resizing
  autoGenerateColumns: false,
  columns:[
  { binding: 'id', header: 'ID', width: 60, allowResizing: false  },
    .
    .
    .
  ],
  itemsSource: data
  });

// 행렬의 크기 변경을 금지합니다.
flexGrid.allowResizing = wijmo.grid.AllowResizing.None;
