검색 텍스트 | Search Text

아래 샘플은 FlexGrid의 속성과 함수들을 응요하여 만들어진 샘플입니다.

'formatHighlightCells' 사용자 정의 이벤트를 바인딩하여 사용자가 텍스트를 입력할 때마다 미리 정의해둔 CSS를 적용하여 동일한 값을 가지고 있는 셀의 Text를 강조합니다. 만약 사용자가 'Enter' 키를 누르거나, 이전('<') / 다음('>') 버튼을 누렀을 경우, 해당 셀로 순차적으로 이동합니다.

인터넷 브라우저와 같은 검색 기능을 구현 하고 싶으실 경우 사용이 가능합니다.
