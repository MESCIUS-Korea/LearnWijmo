필터링 | Filtering

FlexGrid는 기본적으로 Excl와 유사한 UI의 필터링 패널을 제공합니다.
FlexGrid의 필터 패널을 적용하기 위해서는 wijmo.grid.filter.min.js 파일을 header에 선언해야 합니다. 

<script src="scripts/vendor/controls/wijmo.grid.filter.min.js"></script>

이후, FlexgridFilter 함수를 통해, 각 열(컬럼) 해더에 필터 아이콘을 나타나게 할 수 있습니다. 

var filter = new wijmo.grid.filter.FlexGridFilter(flexGrid);

Result와 함게 HTML, JS 코드도 확인이 가능합니다.

검색 필터

사용자가 원하면, 그리드의 collectionView.filter 속성을 직접 사용할 수 있습니다.
이것은 쉽고 매우 유연하지만 필터 UI를 직접 구현해야합니다.

Collection 자체의 Filter 기능을 확장할 경우, 위에서 필터패널을 적용을 위해 선언했던 wijmo.grid.filter.min.js 파일을 header에 선언할 필요는 없습니다.
