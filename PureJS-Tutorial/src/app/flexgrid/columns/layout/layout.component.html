레이아웃 저장 | Persisting the Layout

기본적으로 FlexGrid를 사용하면 사용자가 원하는데로 열의 크기를 조정하고 순서를 변경할 수 있습니다. 하지만, 때때로 이러한 레이아웃 상의 변경사항들을 기억하거나, 원래 모양으로 복원하고 싶을 경우가 있습니다.

이럴 경우, FlexGrid의 columnLayout 속성을 사용하면 사용자가 열 레이아웃을 저장하고 복원 할 수 있습니다.
아래 버튼을 클릭하여 작동 방식을 확인하십시오. 

var layOut = flexGrid.columnLayout;
