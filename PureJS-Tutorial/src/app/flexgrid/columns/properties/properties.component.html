열 속성 | Column Properties

wijmo.grid.Column 클래스에는 각 열의 모양과 동작을 구성하는 데 사용할 수있는 약 40 개의 속성이 있습니다.

이러한 속성은 자동 생성된 열에 대해 자동으로 설정되지만 코드로 열을 만들 때 일반적으로 다음과 같이 설정합니다.

  • binding: 이 열에 표시 될 데이터 항목의 속성을 정의합니다.
  • header: 이 열의 머리글 셀의 내용을 정의합니다.
  • format: 숫자와 날짜를 열의 표시 문자열로 변환하는 데 사용되는 형식 문자열입니다.
  • width: 이 열의 폭 (픽셀 단위)입니다. 'star(별)'크기 조정을 사용할 수도 있습니다.       상대 값으로 열 너비를 지정하십시오 (예 : '2*'는'*'의 두 배).
  • align: 이 열에있는 항목의 가로 맞춤을 정의합니다. 이것은 기본적으로 null로 설정되며, 이는 열의 데이터 유형에 따라 데이터를 정렬합니다. 이 속성을 'left', 'right'또는 'center'로 설정하여 기본값을 재정의 할 수 있습니다.
  • isReadOnly: 이 열을 사용자가 편집 할 수 있는지 여부를 결정합니다. 그리드를 설정하여 편집을 모두 비활성화 할 수 있습니다. isReadOnly 속성을 false로 설정하십시오.
  • isRequired: 이 열의 값을 빈 문자열 또는 null 값으로 설정할 수 있어야 하는지 여부를 결정합니다.
