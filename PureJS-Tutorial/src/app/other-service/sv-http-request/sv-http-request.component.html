wijmo.httpRequest Method

HTTP 요청은 웹 응용 프로그램에서 서버와 데이터를 교환하는 데 사용되는 기본 메커니즘입니다.

많은 JavaScript 툴킷과 프레임 워크에는 HTTP 요청을하기위한 자체 래퍼가 있습니다.예를 들어, Angular에는 $http서비스가 있고 jQuery에는 ajax()메서드가 있습니다.즐겨 사용하는 메소드가 있다면 Wijmo에서 사용할 수 있습니다.

만약, 외부 툴킷 및 프레임 워크에 대한 종속성을 줄이고 싶다면 Wijmo의 간단하고 유연한 httpRequest 메소드를 사용할 수 있습니다.
