색상 | Color

Color 클래스는 CSS 문자열로 지정된 색상을 파싱하고 빨강, 녹색, 파랑 및 알파 채널을 읽기/쓰기 속성으로 표시합니다.

또한 RGB 대신 HSB 및 HSL 색상 모델을 사용하여 색상을 생성하는 데 사용할 수있는 fromHsb 및 fromHsl 메서드와 해당 색상 모델을 사용하여 색상 구성 요소를 가져 오는 getHsb 및 getHsl 메소드도 제공합니다.

Color 클래스에는 HSL 모델을 사용하여 두 색상의 중간 값을 체워 색상을 만드는 interpolate 방법이 있습니다.이 메서드는 애니메이션 메서드로 색상 애니메이션을 만드는 데 특히 유용합니다.
