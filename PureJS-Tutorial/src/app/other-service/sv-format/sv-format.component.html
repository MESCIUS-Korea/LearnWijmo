wijmo.format Method

형식이 지정된 데이터를 기반으로 문자열을 만드는 것은 JavaScript에서 어려울 수 있습니다. ES2016은 템플릿 문자열을 도입함으로써 이러한 한계를 해결했습니다.

유감스럽게도 솔루션에 대한 브라우저 지원은 여전히 제한되어 있으므로 Wijmo는어디서나 작동하는 format 함수 기능을 제공합니다. format 함수는 변수명과 형식 지정자를 포함하는 Placeholder 및 변수를 제공하는 데이터 객체로 format 문자열을 사용합니다.
